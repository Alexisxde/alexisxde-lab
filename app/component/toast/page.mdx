## Component API

### ToastManager

El objeto `toast` devuelto por `useToast()` contiene las siguientes funciones: `default`, `success`, `warning`, `error`. Todas aceptan el mismo objeto de configuración.

| Prop        | Type                       | Default     | Description                                |
| :---------- | :------------------------- | :---------- | :----------------------------------------- |
| `text`      | `string`                   | `undefined` | El mensaje principal de la notificación.    |
| `icon`      | `React.ReactNode`          | `undefined` | Icono opcional que se muestra a la izquierda. |
| `content`   | `string \| React.ReactNode` | `undefined` | Contenido adicional o decorativo a la derecha. |
| `className` | `string`                   | `undefined` | Clases CSS adicionales para el contenedor. |

### useToast Hook

Hook para acceder a las funciones de notificación. Debe usarse dentro de un `ToastProvider`.

```tsx
const { toast } = useToast();

toast.success({ text: "¡Éxito!" });
```
