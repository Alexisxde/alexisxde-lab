# Tab

Responde al cursor con animaciones suaves de escala, amplificando los íconos al pasar el mouse. Esta interacción dinámica refuerza la jerarquía visual y aporta una sensación táctil y envolvente a la navegación.

## Component API

### Tab

| Prop        | Type              | Default     | Description                                |
| :---------- | :---------------- | :---------- | :----------------------------------------- |
| `children`  | `React.ReactNode` | Requerido   | El contenido del `Tab`.                   |
| `className` | `string`          | `undefined` | Clases CSS adicionales para el contenedor. |

### TabAnchor

| Prop        | Type              | Default     | Description                                         |
| :---------- | :---------------- | :---------- | :-------------------------------------------------- |
| `icon`      | `React.ReactNode` | Requerido   | El icono del elemento.                              |
| `title`     | `string`          | `undefined` | Si se deben combinar las props en el elemento hijo. |
| `href`      | `string`          | `undefined` | URL a donde va dirigir a hacer click.               |
| `className` | `string`          | `undefined` | Clases CSS adicionales para el contenedor.          |

### TabButton

| Prop        | Type              | Default     | Description                                         |
| :---------- | :---------------- | :---------- | :-------------------------------------------------- |
| `icon`      | `React.ReactNode` | Requerido   | El icono del elemento.                              |
| `title`     | `string`          | `undefined` | Si se deben combinar las props en el elemento hijo. |
| `onClick`   | `() => unknown`   | `undefined` | function que se ejecutará a hacer click.            |
| `className` | `string`          | `undefined` | Clases CSS adicionales para el contenedor.          |
