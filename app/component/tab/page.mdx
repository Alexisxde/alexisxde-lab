# Tab

El bloque indicador se desliza al tab activo al hacer clic, reforzando la jerarquía visual y la respuesta contextual.

## Component API

### Tab

| Prop           | Type               | Default     | Description                                |
| :------------- | :----------------- | :---------- | :----------------------------------------- |
| `children`     | `React.ReactNode`  | Requerido   | El contenido del `Tab`.                    |
| `defaultValue` | `string \| number` | Requerido   | Valor por defecto del inicio del `Tab`.    |
| `className`    | `string`           | `undefined` | Clases CSS adicionales para el contenedor. |

### TabHeader

| Prop        | Type              | Default     | Description                                |
| :---------- | :---------------- | :---------- | :----------------------------------------- |
| `children`  | `React.ReactNode` | Requerido   | El contenido del `TabHeader`.              |
| `className` | `string`          | `undefined` | Clases CSS adicionales para el contenedor. |

### TabOption

| Prop        | Type               | Default     | Description                                |
| :---------- | :----------------- | :---------- | :----------------------------------------- |
| `children`  | `React.ReactNode`  | Requerido   | El contenido del `TabOption`.              |
| `value`     | `string \| number` | `undefined` | Valor para cada `TabOption`.               |
| `className` | `string`           | `undefined` | Clases CSS adicionales para el contenedor. |
