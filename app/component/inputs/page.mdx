## Component API

### Input

| Prop             | Type      | Default     | Description                                 |
| :--------------- | :-------- | :---------- | :------------------------------------------ |
| `type`           | `string`  | `text`      | El tipo del `Input`.                        |
| `className`      | `string`  | `undefined` | Clases CSS adicionales para el contenedor.  |
| `label`          | `string`  | `undefined` | label que aparecerĂˇ en el input (optional). |
| `optional`       | `boolean` | `true`      | Activa la advertencia roja si es requerida. |
| `disabled`       | `boolean` | `false`     | Desactiva el input.                         |
| `labelClassName` | `string`  | `undefined` | Clases CSS adicionales para el label.       |
